# Chrome-Zoom-Extension
Simple Custom Zoom Extension
# 🔍 trkendimce Zoom (Chrome Extension)

Chrome tarayıcısında varsayılan zoom sınırlarını aşarak **%155, %160, %165 gibi özel zoom değerleri** ayarlamanızı sağlayan basit ve güvenli bir eklenti.

---

## 🚀 Özellikler

* 🎚️ Slider ile hassas zoom ayarı (%50 – %300)
* 🔢 Manuel değer girme (örn: 160 → %160)
* ⌨️ Klavye kısayolları ile hızlı kontrol
* 🔒 Minimum izin (sadece aktif sekmede çalışır)
* 🧩 Hafif ve hızlı (gereksiz kod yok)

---

## ⌨️ Kısayollar

* **Ctrl + Shift + ↑** → Zoom artır
* **Ctrl + Shift + ↓** → Zoom azalt

---

## 📦 Kurulum (Chrome)

1. Bu projeyi indir (Code → Download ZIP)
2. ZIP dosyasını çıkar
3. Chrome'da adres çubuğuna şunu yaz:

   ```
   chrome://extensions
   ```
4. Sağ üstten **Geliştirici modu**nu aç
5. **Paketlenmemiş öğe yükle** butonuna bas
6. Proje klasörünü seç

---

## 🛠️ Kullanım

1. Tarayıcıda eklenti ikonuna tıkla
2. Slider veya input ile zoom değerini seç
3. “Uygula” butonuna bas

---

## 🔐 Güvenlik

Bu eklenti:

* Veri toplamaz
* İnternete bağlanmaz
* Sadece aktif sekmede çalışır
* Minimum Chrome izinlerini kullanır

---

## 📁 Dosya Yapısı

* `manifest.json` → Eklenti ayarları
* `popup.html` → Arayüz
* `popup.js` → Zoom kontrolü
* `background.js` → Klavye kısayolları
* `icon.png` → Eklenti ikonu

---

## ✨ Not

Chrome varsayılan olarak sadece belirli zoom seviyelerine izin verir (%125, %150 vb.).
Bu eklenti sayesinde bu sınırları aşabilirsiniz.
